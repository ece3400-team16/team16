Final Competition

The culmination of the semester

Wanna know more?

Objective

This class was meant to help us all learn more about intelligent physical systems (i.e. robots that can make their own decisions) through making our own. Our robot is named Arduino Schwarznegger, and we wanted him to be able to do the following in time for the final competition:

  • Follow a line
  • Avoid walls
  • Start on a 660Hz tone
  • Solve a maze
  • Display findings to a GUI
  • Avoid other robots
  • Detect treasures

Procedure

Each week there was a new lab or milestone to finish that would help us achieve the objectives above.

  • Lab 1:
    • Learn how to read digital and analog signals
    • Assemble the servos, to be used for the wheels
    • Build the original robot chassis
  • Milestone 1:
    • Have our robot follow a line
    • Improve upon that to have our robot follow a maze grid
  • Lab 2:
    • Understand how to use an FFT with our Arduino
    • Detect a 660Hz acoustic signal
    • Detect a 6kHz IR signal
  • Milestone 2:
    • Add wall sensors to the robot
    • Implement a right-hand follow algorithm to solve the maze
    • Show that robot can detect other robots nearby through IR
  • Lab 3:
    • Store maze information in a space-saving manner
    • Send information across via a radio transmitter on the Arduino to the base station radio receiver
    • Update a maze GUI with information about the robot's location and surroundings
  • Milestone 3:
    • *Actually* solve the maze using a depth-first search algorithm
    • Ensure the robot explores all nodes of the maze
  • Lab 4:
    • Understand how to use an FPGA and camera
    • Use the camera to detect red vs blue color
    • Send that color detection to the Arduino
  • Milestone 4:
    • Detect shapes shown to the camera
    • Send color and shape information about tresures to the Arduino
    • Display and analyze that information accordingly on the GUI
